# Robot Snake demo bundle

This publication preserves selected, inspectable artifacts from an isolated Possible Robot Prototype run. It proves a coherent digital prototype and deterministic local simulation. It does not prove physical locomotion, fabrication readiness, actuator suitability, or functional safety.

The page uses the GLB for an in-browser static CAD inspection surface and the saved MuJoCo GIFs, metrics, trajectories, contract, and receipt as simulation evidence.

`viewer/robot-snake.rrd` is a self-contained Rerun replay of the preserved obstacle-course trajectory. It does not rerun physics. Its source hashes, coverage, and evidence boundary are recorded in `viewer/robot-snake.manifest.json`; local instructions are in `viewer/README.md`.
