# Asset provenance

The local PatchProof launch package uses no external images, fonts, icons, analytics, embeds, trackers, or third-party runtime assets.

| Asset | Origin | License / boundary |
| --- | --- | --- |
| Launch HTML, CSS, and JavaScript | Created locally for this preserved Possible run | Repository-local demonstration; not published by this run. |
| Three Remix preview HTML files | Created locally from the approved PatchProof product truth | Same factual copy and viewport; presentation differs. |
| Remix preview PNGs | Captured locally with Playwright from the three preview HTML files | Evidence screenshots, not product or customer proof. |
| Launch-site desktop/mobile PNGs | Captured locally with Playwright from `launch/site/` | Browser evidence, not a public URL. |
| Product fixture content | Existing tested repository fixtures | Synthetic reproducible evidence, labeled as such. |
| Typography | Operating-system font stacks only | No font files or network requests. |

The fanfold perforations, receipt rules, and signal paths are CSS geometry. They are not copied brand assets or externally sourced illustrations.
