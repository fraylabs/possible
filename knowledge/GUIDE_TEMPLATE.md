# Field guide template

Copy this template into the appropriate folder under `knowledge/pages/`. Replace
every placeholder, add primary or authoritative sources, and link only guides
that are genuinely useful adjacent reading.

```markdown
---
slug: concise-kebab-case-slug
title: Concise guide title
summary: One sentence describing what maintained guidance this guide provides.
tags: [field, subject]
aliases: []
reviewedAt: YYYY-MM-DD
sources:
  - title: Primary or authoritative source
    url: https://example.com/reference
---

## What this helps with

Define the subject and the practical result this guide helps a reader pursue.

## Use this guide when

State the conditions that make the guidance relevant.

## Decisions and inputs

Identify project-specific facts the user or agent must resolve.

## A practical approach

Give sourced guidance without pretending it is a universal project plan.

## Combine with

Link adjacent Possible guides when they add useful context. The consuming agent
decides which guides to combine and which available skills to use.

## Validate the work

Describe concrete checks for the consuming agent to run in its own environment.
Possible does not execute or record those checks.

## Limits and alternatives

Explain when the guidance does not apply and name credible alternatives.
```
