## Technical starting point

Use a one-piece CNC-machined 6061-T6 L-bracket. Fit, measurable geometry, and vibration stiffness justify machining for this five-part lot. Keep bent 5052 sheet as a later cost-focused alternative only if measured loads and allowable squareness support it.

The motor bolts to the upright through four horizontal slots; the bracket base remains fixed to the frame. Thus the motor, shaft, and connector translate together through the belt-tension range without disturbing the frame joint.

## Controlled starting definition

- Units: mm. Quantity: 5. Material: 6061-T6. Finish: deburred, as-machined; no coating on mating faces.
- Provisional envelope: 66 wide × 48 base depth × 58 high, with 6 nominal wall thickness and an inside corner radius of at least 3. The provisional motor-axis height is 29 above the frame face; replace this with the measured belt-line height before release.
- Datums: A = underside contacting the frame; B = motor-contact face; C = left machined side. Adjustment axis X is parallel to A and normal to C.
- Motor interface: four rounded slots, nominally 3.50 +0.15/−0 wide × 16.0 +0.2/−0 overall, centered on a 31.00 basic square at mid-travel and oriented along X. Functional acceptance is at least 12.0 simultaneous travel using four 3.0 pins on a verified 31 mm-square fixture.
- Shaft/pilot window: an X-oriented obround. Its width is measured motor-pilot diameter + 1.0 diametral clearance; its length is that width + 12.0. Enlarge only if the confirmed pulley installation sequence requires passage through the bracket.
- Starting frame interface: four Ø4.5 clearance holes on a provisional 52 × 28 rectangular pattern. Use M4 through-bolts, flat washers, and prevailing-torque nuts; do not rely on threads in only 3 mm of frame plate. Add a backing strip if frame testing shows local flex.
- Initial critical controls: A flatness 0.15; B flatness 0.10; B perpendicular to A within 0.15; motor-slot pattern position 0.20 relative to A|B|C; confirmed motor-axis height ±0.10. General profile dimensions may be ±0.5. These are starting acceptance limits, not unverified provider capabilities.
- Use M3 socket screws with hardened flat washers spanning the slots. Screw length must provide permitted engagement without bottoming. Apply removable threadlocker only if the motor maker permits it; set torque from the confirmed screw grade, thread material, and motor specification. Add witness marks after final torque.

## Later inspection and proof

Inspect all five parts. Verify material identity and revision, then measure A/B flatness, perpendicularity, axis height, frame-hole pattern, slot width/length/position, edge breaks, and burr removal using a surface plate plus height gauge, CMM, or an equivalent documented method. Use the four-pin fixture to prove 12 mm travel without binding.

Dry-assemble one part with the actual motor, pulley, connector, cable, frame, and tools. Require full travel, flush seating without rocking, four screws that start freely, no shaft/pulley/connector interference, and service access at both adjustment limits.

Finally, tension the real belt, torque and witness-mark fasteners, and run a representative vibration/duty test. Reject visible slot slip, witness-mark movement, loosening, cracking, permanent set, connector chafe, or motor-axis displacement beyond the belt system’s user-approved limit. Reinspect after run-in.
